## b2 BBC Micro Emulator

b2 is a cross-platform BBC Micro emulator. Use your Linux PC to play your old BBC games or develop new BBC software.

This snap is a community-maintained unofficial build of [b2](https://github.com/tom-seddon/b2).

[![Get it from the Snap Store](https://snapcraft.io/static/images/badges/en/snap-store-black.svg)](https://snapcraft.io/b2)

## Install

Install on any [supported Linux](https://snapcraft.io/docs/installing-snapd) distribution:

```
snap install b2
```
